# Structure

```
root
┣ config ( generated files from the install )
┣ containers
┃ ┣ apps ( contains running applications like gitea or umami )
┃ ┣ projects ( work in progress projects )
┃ ┗ services ( contains headless services, like the nginx proxy )
┃   ┗ proxy
┣ scripts ( contains all Halloumi scripts )
```
